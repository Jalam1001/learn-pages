LearnPages

This page is served directly from a GitHub repository using GitHub Pages. There is no server, no backend, and no framework involved.

The URL explained

https://jalam1001.github.io/learn-pages/
└──────────────┬──────────────┘ └───────┬───────┘
     GitHub Pages domain            Repository name

What makes this work

  1. This repository is public
  2. It contains an index.html file
  3. GitHub Pages will be enabled for this repo

The key idea

GitHub Pages simply serves static files from a repository. If an index.html exists, it becomes the homepage.

This page exists so the mechanism can be seen, not guessed.
